# Teslo DB Api

1. Clonar el proyecto
2. Instalar el node_modukes
```
yarn install
```
3. Cambiar el nombre del archivo ``` .env.template ``` a ``` .env ``` y configurar sus variables de entorno
5. Levantar la base de datos con docker
```
docker-compose up -d
```
